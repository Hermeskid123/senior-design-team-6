\doxysubsection{Detailed Description}
This class contains driver code for interfacing with the camera. 

\mbox{\hyperlink{class_camera_1_1_camera}{Camera}} is a singleton class so only one istance of a \mbox{\hyperlink{class_camera_1_1_camera}{Camera}} object can be created. This class contains all needed functions to connect to and interface with the camera.
